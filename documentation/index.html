Quick customizations

  1. Change the brand name in index.html (find DEV SPACE).
  2. Replace placeholder images in assets/images/ with your own files (same file names or update src).
  3. To change colors, open assets/css/style.css and modify :root variables (e.g. --bg-gradient-1).
  4. To edit pricing, open index.html and update the pricing cards under #pricing.
